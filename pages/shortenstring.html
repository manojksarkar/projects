ABOUT SHORTEN-STRING

This site has two part:

Encode : It takes a string and convert into a shorter string. The input strings are alphabets only and provided in small case only (for eg helloworld). The encoded string will be smaller in length(which consists alphanumeric) It is gauranted that no two different input strings result in same shortened output string.

Decode : By giving the encoded string it will decode the original string.
